<h1>GimmeAName - FoundryVTT Module</h1>

<h2>Module Description</h2>

<p>
GimmeAName is a FoundryVTT module designed to inspire Game Masters by providing a tool to quickly generate names for various elements within their campaigns.  This module uses IntegrateAI to create unique and evocative names for NPCs, monsters, items, locations, organizations, and more.
</p>

<h2>Key Features</h2>

<ul>
  <li><b>AI-Powered Generation:</b> Uses IntegrateAI to generate names, saving you time and sparking your creativity.</li>
  <li><b>Customizable Parameters:</b>  Fine-tune the name generation process with options like:
    <ul>
      <li><b>Number of Names:</b> Specify how many names you want to generate.</li>
      <li><b>Additional Description:</b>  Provide context to guide the generation (e.g., "a dark and foreboding castle").</li>
      <li><b>Examples:</b>  Give examples of names you like.</li>
      <li><b>Excluded Names:</b>  Dissuade it from generating names you don't want.</li>
    </ul>
  </li>
  <li><b>Versatile for Various Elements:</b>  Generate names for NPCs, monsters, items, locations, organizations, and more.</li>
</ul>

<h2>How to Use</h2>

<p>
This module is designed to be intuitive and easy to use.  The core functionality is centered around the <code>generateName</code> function, which you can call with a prompt.  The module provides several pre-defined functions for specific types of elements (e.g., <code>generateNPCNames</code>, <code>generateMonsterNames</code>).
</p>


<p>
Experiment with different prompts and parameters to find the perfect names for your campaign!
</p>
