## Module Description

GimmeAName is a FoundryVTT module designed to inspire Game Masters by providing a tool to quickly generate names for various elements within their campaigns. This module uses IntegrateAI to create unique and evocative names for NPCs, monsters, items, locations, organizations, and more.

## Key Features

- **AI-Powered Generation:** Uses IntegrateAI to generate names, saving you time and sparking your creativity.
- **Customizable Parameters:** Fine-tune the name generation process with options like:
  - **Number of Names:** Specify how many names you want to generate.
  - **Additional Description:** Provide context to guide the generation (e.g., "a dark and foreboding castle").
  - **Examples:** Give examples of names you like.
  - **Excluded Names:** Prevent the generator from returning names you donâ€™t want.
- **Versatile for Various Elements:** Generate names for NPCs, monsters, items, locations, organizations, and more.

## How to Use

This module is designed to be intuitive and easy to use. The core functionality is centered around the `generateName` function, which you can call with a prompt. The module also provides several pre-defined helper functions for specific element types (e.g., `generateNPCNames`, `generateMonsterNames`).

Experiment with different prompts and parameters to find the perfect names for your campaign!
